# grep

```grep -i "somefilename"```

* ```-i``` - ignore case
* ```-E``` for regex

```grep -irl "text"```
<br>
```grep -r "password" /home/username```
<br>
```grep -E 'hell?o' file.txt```
<br>
```grep -E '??bla.??' file.txt```
<br>
```grep -E '??[1-3]*' file.txt```
<br>
```grep -E '*[!1-3].???' file.txt```
